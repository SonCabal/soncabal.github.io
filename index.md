## 📖 About this software

Data lives in many places. You store information in analytics tools, artificial intelligence platforms, and business intelligence software. Often, these tools do not speak the same language. This causes errors and makes your work harder.

Ossie solves this problem. It acts as a translator for your data systems. It creates a single record that every tool understands. By using this standard, your data stays consistent from the source to the final report. You spend less time fixing data and more time using it. Our project bridges the gap between different software vendors. It provides a neutral home for your semantic metadata.

## 🛠️ How to use the application

Once you open the software, you see a clean dashboard. You can link your data sources by selecting the "Add Source" button. The application scans your connected files and creates a map of your metadata.

## 🛡️ Troubleshooting common issues

Most users have a smooth experience. If you hit a wall, check the tips below:

* **I cannot open the file:** Ensure you downloaded the version for Windows. Sometimes browsers block files by default. Check your browser security settings if the file does not save.
* **The software crashes on start:** Check that you have the latest updates for Windows. Sometimes a restart helps clear stuck background processes.
* **My data does not sync:** Verify that your internet connection is active. Ensure the path to your data file is correct. If the data resides on a network drive, check that you have the right permissions to access that folder.
* **The screen looks small or distorted:** Check your display scaling settings in Windows. Ossie works best at 100% display scaling.

## 📋 Manage system settings

You can change how the software works in the Settings menu. Select the gear icon in the top right corner. Here, you can change the default location for your metadata files. You can also toggle the "Auto-Sync" feature. When this is on, the software looks for changes to your data every hour. When this is off, you must press the "Refresh" button to update your records.
